Happenly

Happenly is a modern web application that helps users discover, filter, and visualize events in their city, offering a personalized and interactive experience. The app features offline access, customizable event listings, and installable PWA functionality for seamless use across devices. Built with React, Vite, and robust testing tools, Happenly emphasizes accessibility, performance, and user-centric design.
